# Meliora VC - Server-Side Voice Chat Plugin

Complete Minecraft server voice chat plugin with character modes and private groups.

## Quick Start

1. Extract this ZIP
2. See `BUILD_INSTRUCTIONS.md` for how to build
3. Copy resulting JAR to plugins folder
4. Restart server

## Features

✅ Universal access (all players can use)
✅ Proximity-based chat (100m, 25m whisper, 250m yell)
✅ Private voice groups with global chat
✅ Character modes (IC/OOC)
✅ Admin voice controls
✅ GUI-based management
✅ Keybind support (V, G, I, O, M, N, Y)

## System Requirements

- Java 11 or higher
- Maven (for building)
- Minecraft server 1.18+

## GitHub Setup

1. Create new repository on GitHub
2. Upload these files
3. GitHub Actions will build automatically
4. Download JAR from Actions artifacts

## Support

- In-game: `/vchelp`
